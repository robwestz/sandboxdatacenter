## SEO PATTERNS

### KEYWORD-CLUSTER (Keyword Research)
```
1. SEED-ANALYZER: Understand core topic
2. EXPANDER: Generate related terms, questions, long-tail
3. INTENT-MAPPER: Classify each by search intent
4. CLUSTERER: Group semantically related keywords
5. PRIORITIZER: Score by opportunity (volume × inverse difficulty)
6. OUTPUT: Clustered keyword map with priorities
```

### CONTENT-GAP (Competitor Analysis)
```
1. SERP-ANALYZER: Map what top 10 cover
2. TOPIC-EXTRACTOR: Identify all subtopics across results
3. COVERAGE-SCORER: Rate coverage per subtopic per result
4. GAP-FINDER: Find low-coverage, high-relevance topics
5. OPPORTUNITY-RANKER: Score gaps by potential
6. BRIEF-GENERATOR: Create briefs for top gaps
```

### CONTENT-BRIEF (Brief Generation)
```
1. INTENT-ANALYZER: Define primary + secondary intent
2. SERP-RESEARCHER: Analyze top-ranking content structure
3. TOPIC-MAPPER: Required topics, entities, questions
4. STRUCTURE-DESIGNER: Optimal headings, flow
5. DIFFERENTIATOR: Unique angle identification
6. BRIEF-COMPILER: Assemble tactical brief
```

### OPTIMIZE-LOOP (Content Optimization)
```
1. CONTENT-ANALYZER: Score existing content
2. GAP-IDENTIFIER: What's missing vs. SERP leaders?
3. RECOMMENDATION-ENGINE: Specific improvements
4. PRIORITY-RANKER: Order by impact
5. OUTPUT: Prioritized optimization checklist
```

### LINK-ARCHITECT (Internal Linking)
```
1. SITE-MAPPER: Understand content relationships
2. AUTHORITY-SCORER: Identify pillar/cluster pages
3. GAP-FINDER: Missing semantic connections
4. ANCHOR-GENERATOR: Natural anchor text suggestions
5. PRIORITY-RANKER: Highest-impact links first
6. OUTPUT: Link recommendations with anchors
```

## SEO VARIABLE MARRIAGES

| Variable A | Variable B | Marriage Rule |
|------------|------------|---------------|
| Keyword intent | Content type | Informational → Guide, Commercial → Comparison |
| Competition | Content depth | High comp → More comprehensive |
| Search volume | Investment | High volume → More iterations |
| Content type | Structure | How-to → Steps, Review → Pros/cons |
| Target audience | Language | B2B → Professional, B2C → Accessible |
| Funnel stage | CTA | TOFU → Learn more, BOFU → Buy now |
