SETUP INSTRUCTIONS:

1. Run setup (ONE TIME ONLY):

   Windows:
   > python setup.py

   Linux/Mac:
   $ python3 setup.py

   This takes 10-30 seconds and installs dependencies.

2. Start building:

   Windows:
   > factory.bat "Build a todo app"

   Linux/Mac:
   $ ./factory.sh "Build a todo app"

REQUIREMENTS:
- Python 3.8 or higher
- Internet connection (first-time setup only)
- ~50 MB disk space (after setup with venv)
