## THE FIVE CYCLES

### CYKEL 0: ORIGINAL ANALYSIS

```
1. PRODUCT ARCHAEOLOGY
   - Vad gör produkten?
   - Vem använder den?
   - Hur tjänar de pengar?
   - Vad är deras historia? (varför ser den ut så?)

2. SUCCESS FACTORS
   - Varför vann de? (timing? execution? product? distribution?)
   - Vad gör de BÄTTRE än alternativ?
   - Vad är deras "secret sauce"?

3. CORE VS ACCIDENTAL
   - CORE: Utan detta är det inte denna produkt
   - ACCIDENTAL: Historiska val som kunde varit annorlunda
   - NICE-TO-HAVE: Features som inte är kärnan

4. USER MENTAL MODEL
   - Hur tänker användare om produkten?
   - Vilka "jobs to be done" löser den?
   - Vad är användarnas workflow?
```
